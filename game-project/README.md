# TypeScript RPG Game - Consolidation Project

## Overview
Build a text-based RPG game that demonstrates all major TypeScript concepts in a fun, practical way. Estimated completion time: 2-3 hours.

## Game Features
- Character creation with different classes
- Turn-based combat system
- Inventory management
- Level progression
- Save/load game state

## TypeScript Concepts Covered
- ✅ **Classes & Inheritance** - Character classes, items
- ✅ **Interfaces & Types** - Game state, combat actions
- ✅ **Generics** - Inventory system, event handling
- ✅ **Union Types** - Action types, item categories
- ✅ **Abstract Classes** - Base character, item types
- ✅ **Decorators** - Logging, validation
- ✅ **Modules** - Game systems organization
- ✅ **Async/Await** - Save/load operations
- ✅ **Utility Types** - Configuration, state management

## Project Structure
```
game-project/
├── src/
│   ├── characters/     # Character classes
│   ├── items/         # Item system
│   ├── combat/        # Battle mechanics
│   ├── game/          # Core game logic
│   └── utils/         # Utilities & decorators
├── data/              # Game data files
└── saves/             # Save game files
```

## Getting Started
1. Read through the base classes
2. Implement the TODOs step by step
3. Test each feature as you build
4. Extend with your own ideas!

## Run the Game
```bash
cd game-project
npx ts-node src/main.ts
```
